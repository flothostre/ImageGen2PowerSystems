# AttEMT - *Attempting to simulate EMT phenomena using the Attention mechanism*

## Rationale
This is the codebase for the paper submission to **PSCC 2026**. In detail, we aim to predict power system dynamics in real time (or even faster) leveraging the multi-head attention mechanism coupled with vector quantization.
